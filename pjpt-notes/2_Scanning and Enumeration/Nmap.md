**NMAP  →  Network Mapper**

command :  nmap -T4 -p- -A <victim's_ip>

T1 - T4  →  speed of the scanning
p  →  ports
-p-  →  represents all ports (65535)
-p 80,443  →  separate ports can be defined
If the flag is not mentioned then nmap scans the first 1000 ports
